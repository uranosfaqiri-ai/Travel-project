# Travel Planner Website ✈️

A modern travel booking website built using **HTML, CSS, and JavaScript**.

## Features
- Responsive navigation menu
- Popular destinations section
- Travel packages
- Client reviews slider
- Booking form
- Beautiful animations using ScrollReveal

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Swiper.js
- ScrollReveal
